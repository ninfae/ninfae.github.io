# Ninfa Espinoza — Artist Website

Sitio web personal de Ninfa Espinoza, artista visual hondureña.
Construido con HTML, CSS y JavaScript, sin frameworks.

## Archivos

| Archivo | Qué contiene |
|---|---|
| `index.html` | El contenido y la estructura: textos, obras, CV |
| `style.css` | Todo lo visual: colores, tamaños, espacios |
| `script.js` | La interactividad: visor de imágenes, menú |
| `images/works/` | Fotos de las obras |
| `images/about/` | Retrato y fotos de exposiciones |

## Cómo verlo

Abre `index.html` con doble clic. Se abre en el navegador.

## Cómo cambiar algo

- **Un texto** → búscalo en `index.html` y escribe encima
- **Un color** → cambia los valores en `:root`, arriba de `style.css`
- **Agregar una obra** → copia un bloque `<figure class="work">` completo
  en `index.html` y cambia la imagen y los textos

Después de cualquier cambio, guarda el archivo y recarga el navegador
con `Ctrl + R`.

## Guardar los cambios en Git

```bash
git status              # ver qué cambió
git add .               # marcar todo para guardar
git commit -m "Cambié X"  # guardar con una nota
git push                # subir a GitHub
```
